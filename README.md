# ClutchCue – Riot Review Mockup

This repository contains an interactive product mockup for **ClutchCue**, a planned public VALORANT highlight recording and post-game coaching application.

The mockup demonstrates:

- automatic highlight organization;
- post-match impact analysis;
- personal playstyle insights;
- long-term progress comparisons; and
- optional post-match AI coaching.

All match data shown in this mockup is simulated. The mockup does not connect to VALORANT, Riot Games, Overwolf, or any external service.

## View locally

Open `index.html` in a modern browser. No installation, build command, or dependency is required.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Push the contents of this repository to the `main` branch.
3. Open **Settings → Pages** in GitHub.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder.
6. Save and wait for GitHub to provide the public URL.

The page includes a `noindex` directive to discourage search-engine indexing. Anyone with the URL can still access it, which allows Riot's reviewers to inspect the user flow without creating an account or downloading an application.
